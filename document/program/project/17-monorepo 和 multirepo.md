# **monorepo** 和 **multirepo**(**polyrepo**)

它们都是项目管理的一种方式

- **multirepo**(**polyrepo**) 就是将不同的项目放在不同的 git 仓库维护。
- **monorepo** 将多个项目放在同一个 git 仓库中维护。
