# commitlint

## Prompt 提示器

提示器类似 commitizen 提供commit 提示工具。

```json
npm i -D  
// commitlint 命令行
@commitlint/cli  
// 预设规则
@commitlint/config-conventional 
// 提示信息命令行
@commitlint/prompt-cli
```
