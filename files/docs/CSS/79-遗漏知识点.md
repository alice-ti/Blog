## `<template>` 元素

> template 元素用于保存 HTML 模板，同时他里面的内容是不可见的。
> 声明后可通过 JS 访问里面的内容，复制或渲染.

## `<picture>` 标签

```html
<picture>
  <source srcset="large.jpg" media="(min-width: 800px)" />
  <source srcset="medium.jpg" media="(min-width: 500px)" />
  <img src="small.jpg" />
</picture>
```
