> place-items 是一个简写属性 ，它允许你在相关的布局（如 **Grid** 或 **Flexbox**）中可以同时沿着块级和内联方向对齐元素 (例如：**align-items** 和 **justify-items** 属性) 。如果未提供第二个值，则第一个值作为第二个值的默认值。
