## Next.js 路由

在 **Next.js 12（及以下**）对应的路由系统，是所有路由文件都写在 **pages** 目录下，每个文件都会生成一个路由

相比于 **Next 12** ,**Next 13** 一个比较大的改动是基于文件的路由系统，增加了一个 **app** 目录，每一层路由必须建一个文件夹，在该文件夹中建立 **page.tsx** 作为该路由主页面
